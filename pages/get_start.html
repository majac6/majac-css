Requirement

  • NodeJS(NPM)

How to get

            
  npm i @majac/css --save

Import

If you use SCSS/SASS, you can use @import 

              
  @charset 'utf-8';
  @import 'build/forSASS.scss';
  @import 'build/majac.css';

If you use CSS, set in your <head> 

              
  <link rel="stylesheet" href="your_path/node_modules/@majac/css/build/majac.css">
